## 摘要
深度研究智能体 (DRAs) 将大语言模型 (LLM) 的推理能力与外部工具相结合。记忆系统使 DRAs 能够利用历史经验，这对于高效推理和自主演化至关重要。现有方法依赖于从记忆中检索相似轨迹以辅助推理，但面临记忆演化效率低下以及存储和检索成本不断增加的关键局限。为解决这些问题，我们提出了一种新型的记忆智能体 (MIA) 框架，该框架由“管理者-规划者-执行者” (Manager-Planner-Executor) 架构组成。记忆管理者 (Memory Manager) 是一个非参数化记忆系统，可以存储压缩后的历史搜索轨迹；规划者 (Planner) 是一个参数化记忆智能体，能够为问题生成搜索计划；执行者 (Executor) 是另一个在搜索计划指导下进行信息搜索和分析的智能体。为了构建 MIA 框架，我们首先采用交替强化学习范式来增强规划者与执行者之间的协作。此外，我们使规划者能够在测试时学习 (test-time learning) 过程中持续演化，在不中断推理过程的情况下，随推理同步进行即时更新。此外，我们在参数化记忆与非参数化记忆之间建立了双向转换循环，以实现高效的记忆演化。最后，我们引入了反思和无监督判断机制，以提升在开放世界中的推理和自我演化能力。在 11 个基准测试上的广泛实验证明了 MIA 的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

# 论文总结：Memory Intelligence Agent (MIA)

## 1. 核心问题与整体含义（研究动机和背景）
深度研究智能体（Deep Research Agents, DRAs）通过结合 LLM 推理与外部搜索工具来处理复杂任务。然而，现有的记忆系统面临三大挑战：
*   **存储与检索效率低下**：长上下文记忆会稀释模型注意力，引入噪声，且随着交互轮数增加，存储和计算成本呈指数级增长。
*   **记忆演化不力**：现有方法多为简单的检索增强（RAG），缺乏将经验转化为模型内在能力（参数化）的机制。
*   **规划与执行脱节**：预训练模型作为规划者时缺乏任务特定训练，导致生成的计划难以被执行者有效遵循。

本论文提出了 **MIA (Memory Intelligence Agent)** 框架，旨在通过一种仿生架构实现记忆的高效压缩、参数化内化以及在推理过程中的实时自我演化。

## 2. 方法论：核心思想与关键技术
MIA 采用了 **Manager-Planner-Executor（管理者-规划者-执行者）** 的解耦架构：

*   **核心架构**：
    *   **Memory Manager (管理者)**：非参数化记忆系统。负责将冗长的搜索轨迹压缩为抽象的“工作流（Workflow）”和“图像描述（Caption）”，并基于语义相似度、价值奖励和频率奖励进行混合检索。
    *   **Planner (规划者)**：参数化记忆智能体。负责分析检索到的历史案例，为当前问题制定分步搜索计划，并具备“反思-重规划”能力。
    *   **Executor (执行者)**：操作终端。在计划指导下通过 ReAct 循环调用工具（如搜索引擎）并分析结果。

*   **关键技术细节**：
    *   **交替强化学习 (Alternating RL)**：基于 **GRPO (Group Relative Policy Optimization)** 算法。第一阶段训练执行者遵循计划；第二阶段在冻结执行者的基础上，训练规划者吸收记忆并优化规划逻辑。
    *   **测试时学习 (Test-Time Learning, TTL)**：在推理阶段，规划者通过在线 RL 进行即时参数更新，将当前探索的成功/失败经验实时内化。
    *   **双向记忆转换**：实现了非参数化记忆（外部存储的工作流）与参数化记忆（模型权重）的循环演化。
    *   **无监督评审框架**：模拟学术论文评审机制（Reviewer-AC），通过逻辑、事实、结果有效性三个维度的自动化评审，在无标注环境下实现自主进化。

## 3. 实验设计
*   **数据集/场景**：涵盖了 11 个基准测试。
    *   **多模态任务**：FVQA, InfoSeek, SimpleVQA, LiveVQA, MMSearch 以及两个自建的科学/新闻领域数据集（In-house 1 & 2）。
    *   **纯文本任务**：HotpotQA, 2WikiMultiHopQA, SimpleQA, GAIA。
*   **对比方法**：
    *   **前沿闭源模型**：GPT-4o, GPT-5.4, Gemini-2.5-Pro, Gemini-3-Flash, Claude-Sonnet-4.6。
    *   **开源 SOTA 智能体**：MMSearch-R1, DeepMMSearch-R1, WebWatcher, Deepeyes2。
    *   **记忆框架**：RAG, Mem0, A-Mem, ReasoningBank, ExpeL, Memento。

## 4. 资源与算力
*   **硬件环境**：使用了 **8 张 GPU** 进行强化学习训练。
*   **软件框架**：基于 **veRL** 强化学习框架，使用 **SGLang** 进行异步 Rollout（采样）。
*   **模型配置**：执行者初始化为 **Qwen2.5-VL-7B**，规划者初始化为 **Qwen3-8B**，管理者/评审员使用 **Qwen3-32B**。
*   **训练细节**：执行者训练 8 个 Epoch，规划者训练 4 个 Epoch，学习率均为 $1 \times 10^{-6}$。

## 5. 实验数量与充分性
论文实验设计非常充分且具有高度的客观性：
*   **多维度评估**：不仅在 11 个数据集上验证了准确率，还进行了消融实验（验证 Manager、Planner、Reflect、TTL 各组件贡献）、工具调用分布分析、训练曲线分析。
*   **跨模型泛化**：验证了 MIA 框架在 GPT-5.4 等闭源模型上的适配性，证明了其作为“插件”提升顶尖模型能力的潜力。
*   **演化验证**：通过多轮迭代实验展示了模型在无监督环境下的持续进步能力。

## 6. 主要结论与发现
*   **性能飞跃**：MIA 显著提升了现有 SOTA 模型的表现。例如，将 GPT-5.4 在 LiveVQA 上的表现提升了 9%。
*   **小模型逆袭**：搭载 MIA 的 **7B 执行者** 在平均性能上超过了 **32B 的同系列模型** 达 18%，甚至逼近或超过了部分闭源巨型模型。
*   **记忆噪声抑制**：实验证明传统的长上下文记忆（如 RAG, Mem0）在复杂研究任务中往往因引入噪声而导致性能下降，而 MIA 的压缩与规划机制有效解决了此问题。
*   **自主进化**：在无监督设置下，MIA 通过自我评审机制实现了与有监督设置相当的性能提升。

## 7. 优点
*   **架构创新**：Manager-Planner-Executor 的解耦设计非常符合人类“经验总结-制定计划-动手执行”的认知逻辑。
*   **实时性**：TTL 机制允许模型在推理时“边做边学”，无需离线重新训练。
*   **高效压缩**：将冗长的搜索轨迹转化为结构化工作流，极大地降低了存储压力和检索干扰。
*   **无监督潜力**：提出的 Reviewer-AC 评审框架为智能体在缺乏人类反馈的开放世界中生存提供了可行方案。

## 8. 不足与局限
*   **推理延迟**：由于引入了规划、执行、反思、重规划以及在线参数更新，单次推理的时间成本（Latency）显著高于直接回答。
*   **算力门槛**：虽然推理时可以使用 7B 模型，但 TTL 过程仍涉及梯度计算和参数更新，对部署环境的显存和算力有一定要求。
*   **工具依赖**：目前主要针对搜索类任务，对于需要极高实时交互或物理模拟环境的任务，其规划机制的泛化性尚待验证。
*   **评审偏差**：无监督演化依赖于 32B 模型的评审质量，若评审模型本身存在偏见或幻觉，可能会引导智能体向错误方向演化。
