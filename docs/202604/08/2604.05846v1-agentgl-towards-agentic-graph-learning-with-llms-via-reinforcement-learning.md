## 摘要
大语言模型（LLMs）日益依赖智能体能力——如迭代检索、工具使用和决策——来克服静态参数化知识的局限性。然而，现有的智能体框架将外部信息视为非结构化文本，未能利用现实世界数据中固有的拓扑依赖关系。为了弥补这一差距，我们引入了智能体图学习（Agentic Graph Learning, AGL），这是一种将图学习重新定义为拓扑感知导航与基于 LLM 推理的交替过程的新范式。具体而言，我们提出了 AgentGL，这是首个由强化学习（RL）驱动的 AGL 框架。AgentGL 为 LLM 智能体配备了用于多尺度探索的图原生工具，通过搜索约束思维来规范工具使用以平衡准确性与效率，并采用图条件课程强化学习策略，在无需逐步监督的情况下稳定长程策略学习。在多种文本属性图（TAG）基准测试和多个 LLM 主干模型上，AgentGL 的表现显著优于强大的 GraphLLMs 和 GraphRAG 基准模型，在节点分类和链路预测中分别实现了高达 17.5% 和 28.4% 的绝对提升。这些结果表明，AGL 是使 LLM 能够在复杂关系环境中自主导航和推理的一个极具前景的前沿方向。代码已公开于 https://github.com/sunyuanfu/AgentGL。

## Abstract
Large Language Models (LLMs) increasingly rely on agentic capabilities-iterative retrieval, tool use, and decision-making-to overcome the limits of static, parametric knowledge. Yet existing agentic frameworks treat external information as unstructured text and fail to leverage the topological dependencies inherent in real-world data. To bridge this gap, we introduce Agentic Graph Learning (AGL), a paradigm that reframes graph learning as an interleaved process of topology-aware navigation and LLM-based inference. Specifically, we propose AgentGL, the first reinforcement learning (RL)-driven framework for AGL. AgentGL equips an LLM agent with graph-native tools for multi-scale exploration, regulates tool usage via search-constrained thinking to balance accuracy and efficiency, and employs a graph-conditioned curriculum RL strategy to stabilize long-horizon policy learning without step-wise supervision. Across diverse Text-Attributed Graph (TAG) benchmarks and multiple LLM backbones, AgentGL substantially outperforms strong GraphLLMs and GraphRAG baselines, achieving absolute improvements of up to 17.5% in node classification and 28.4% in link prediction. These results demonstrate that AGL is a promising frontier for enabling LLMs to autonomously navigate and reason over complex relational environments. The code is publicly available at https://github.com/sunyuanfu/AgentGL.

### 2. 方法论
AgentGL 框架由三个核心支柱组成：
*   **图原生搜索工具（GNS Tools）**：为智能体配备了四种多尺度探索工具：
    *   **1-hop/2-hop 邻域搜索**：基于拓扑连接和语义相关性（加权融合）检索局部证据。
    *   **结构显著性搜索（SS）**：利用 PageRank 分数检索全局关键节点。
    *   **图稠密搜索（DENSE）**：利用语义嵌入相似度跨越拓扑孤岛检索相关节点。
*   **两阶段强化学习策略**：
    *   **阶段 1：策略引导（Bootstrapping）**：通过奖励塑造（格式奖励、准确性奖励、工具覆盖奖励）训练智能体掌握工具调用的基本能力。
    *   **阶段 2：缓解搜索过度（MSO）**：引入“搜索约束思维”，通过回顾性终止触发和认知密度正则化，迫使智能体在保证准确性的前提下减少冗余搜索，提升效率。
*   **图条件课程学习（GCCL）**：
    *   根据图的固有属性（如节点分类中的同质性和度数、链路预测中的语义-标签一致性）量化样本难度，实现从易到难的平滑训练，稳定长程策略的收敛。

### 3. 实验设计
*   **数据集**：涵盖 3 个领域的 7 个 TAG 基准数据集：
    *   引用网络：OGB-Arxiv, PubMed, Arxiv-2023。

    *   社交网络：Reddit。
*   **任务场景**：节点分类（NC）和链路预测（LP），包括**域内评估**和**零样本迁移（Zero-shot Transfer）**。

    *   **通用智能体搜索**：Search-R1, Search-O1。

### 4. 资源与算力
*   **硬件环境**：使用了 **8 张 NVIDIA H100-80G-SXM5 GPU**。
*   **软件框架**：基于 Open-RLHF 进行强化学习训练。
*   **模型规模**：测试了 Qwen2.5-3B 和 7B 两种参数规模的主干模型。
*   **训练细节**：学习率为 2e-6，最大序列长度 1600，每个 Prompt 采样 16 个 Rollout。

### 5. 实验数量与充分性
*   **实验规模**：论文进行了大规模的跨领域测试，涵盖了从 3B 到 7B 的模型缩放实验。
*   **消融实验**：非常详尽，分别验证了两阶段训练（GNSPB & MSO）、奖励项（rCOV, rdepth）、搜索约束组件（RTT, CDR）以及课程学习（GCCL）的有效性。
*   **客观性与公平性**：所有 LLM 基座方法均使用相同的主干模型；针对非图原生智能体（如 Search-R1），作者将其搜索空间限制在图节点内以确保对比的公平性。实验结果显示了显著的性能提升，具有较强的说服力。

### 6. 主要结论与发现
*   **性能飞跃**：AgentGL 在节点分类上比基准提升高达 **17.5%**，在链路预测上提升高达 **28.4%**。
*   **自适应性**：相比静态上下文注入，AgentGL 的交替推理-搜索循环在零样本迁移场景下表现出更强的鲁棒性。
*   **效率平衡**：通过第二阶段的 MSO 训练，模型在减少约 **17.5%** 工具调用的同时，准确率反而有所提升，证明了“少即是多”的搜索策略。
*   **算法偏好**：发现 GRPO 算法在节点分类上略优，而 REINFORCE++ 在链路预测上表现更强。

### 7. 优点
*   **范式创新**：首次将图学习定义为受约束的智能体决策过程，打破了静态图表征的限制。
*   **工具设计完备**：结合了局部拓扑、全局结构和语义相关性，覆盖了图探索的多个维度。
*   **训练稳定性**：通过 GCCL 课程学习解决了强化学习在复杂图环境中难以收敛的问题。
*   **可解释性**：通过 `<think>` 块展示了智能体如何根据已搜集证据调整后续搜索计划。

### 8. 不足与局限
*   **模态限制**：目前仅支持文本属性图，尚未扩展到包含图像或视频的多模态属性图。
*   **训练成本**：虽然推理效率得到了优化，但强化学习（尤其是多阶段 RL）的训练过程对算力要求较高。
*   **数据分配敏感性**：MSO 阶段的稳定性高度依赖于两个阶段之间训练数据的合理分配。
*   **图密度挑战**：在极度稠密的图中，工具返回的信息可能会因上下文窗口限制而导致信息丢失。
