## 摘要
深度研究智能体（DRA）将大语言模型（LLM）的推理能力与外部工具相结合。记忆系统使 DRA 能够利用历史经验，这对于高效推理和自主演化至关重要。现有方法依赖于从记忆中检索相似轨迹以辅助推理，但面临记忆演化效率低下以及存储和检索成本不断增加的关键局限。为解决这些问题，我们提出了一种新型的记忆智能体（MIA）框架，该框架由“管理者-规划者-执行者”（Manager-Planner-Executor）架构组成。记忆管理者是一个非参数化记忆系统，可以存储压缩后的历史搜索轨迹。规划者是一个参数化记忆智能体，能够为问题生成搜索计划。执行者是另一个智能体，可以在搜索计划的指导下搜索并分析信息。为了构建 MIA 框架，我们首先采用交替强化学习范式来增强规划者与执行者之间的协作。此外，我们使规划者能够在测试时学习（test-time learning）过程中持续演化，在不中断推理过程的情况下，随推理同步进行实时更新。此外，我们在参数化记忆与非参数化记忆之间建立了双向转换循环，以实现高效的记忆演化。最后，我们引入了反思和无监督判断机制，以提升在开放世界中的推理和自我演化能力。在 11 个基准测试上的广泛实验证明了 MIA 的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

*   **存储与检索成本：** 随着任务增加，记忆库无限膨胀，导致计算效率低下。
*   **缺乏过程导向记忆：** 现有系统多存储事实性知识（What），而忽略了如何解决问题的策略性经验（How）。
**MIA 的核心目标**是构建一个类脑的记忆架构，通过解耦规划与执行，实现记忆的高效压缩、参数化内化以及在无监督环境下的自主进化。

### 2. 方法论：核心思想与关键技术
MIA 提出了 **Manager-Planner-Executor（管理者-规划者-执行者）** 架构：
*   **Memory Manager（非参数化记忆）：** 模拟海马体，将冗长的搜索轨迹压缩为结构化的“工作流摘要”（Workflow Summaries），存储高价值的成功与失败案例。
*   **Planner（参数化记忆智能体）：** 充当大脑，根据检索到的相似案例生成搜索计划。通过训练，它能将外部经验内化为模型参数。
*   **Executor（执行智能体）：** 负责具体工具调用（如搜索引擎）和信息分析，遵循 Planner 的指令。
*   **关键技术细节：**
    *   **交替强化学习（Alternating RL）：** 使用 GRPO 算法，第一阶段训练 Executor 遵循计划，第二阶段训练 Planner 吸收记忆并进行反思。
    *   **测试时学习（Test-Time Learning, TTL）：** 在推理过程中实时更新 Planner 参数，实现“边做边学”。
    *   **双向转换循环：** 成功的轨迹被压缩进非参数记忆，同时通过 RL 转化为 Planner 的参数化知识。
    *   **无监督进化机制：** 引入类似学术评审的“审稿人-领域主席”机制，通过逻辑、事实和有效性三个维度的自我评判，在没有标准答案的情况下实现自我进化。

### 6. 主要结论与发现
*   **性能飞跃：** MIA 显著提升了现有 SOTA 模型的表现。例如，将 GPT-5.4 在 LiveVQA 上的表现提升了 9%。
*   **以小博大：** 基于 **7B** 规模 Executor 的 MIA 框架，平均性能超过了 **32B** 的基座模型 18%，甚至逼近 Gemini-3-Flash 等顶级闭源模型。
*   **记忆有效性：** 传统的长上下文记忆（如 RAG）在复杂任务中往往因噪声导致性能下降，而 MIA 的压缩工作流和参数化更新能持续带来正向收益。
*   **自主进化：** 在无监督设置下，MIA 通过多次迭代（Epoch 1-3），性能呈现稳步上升趋势，证明了其在开放世界中的生存能力。

### 7. 优点
*   **架构创新：** 成功解决了智能体记忆“存不下、搜不准、用不好”的痛点，通过参数化与非参数化的结合实现了记忆的高效利用。
*   **实时演化：** TTL 机制允许智能体在不停止服务的情况下根据新环境自我优化。
*   **通用性强：** 实验证明该框架不仅适用于开源模型，也能作为插件提升闭源 API 的表现。
*   **无监督闭环：** 提出的类学术评审机制为缺乏标注数据的真实场景提供了可行的进化路径。

### 8. 不足与局限
*   **计算开销：** 虽然推理时记忆检索变快了，但 TTL 阶段需要进行多次采样（rollout）和参数更新，这会显著增加推理时的延迟和算力消耗。
*   **判断器依赖：** 系统的上限在很大程度上受限于作为“判断器”和“管理器”的大规模模型（如 Qwen3-32B），如果判断器产生幻觉，可能会导致错误的记忆内化。
*   **应用领域：** 目前主要集中在“深度研究”（信息检索与问答）任务，在需要实时物理交互或极高实时性的场景下尚未验证。
