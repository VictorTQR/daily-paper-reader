## 摘要
智能体技能是程序性知识和可执行资源的结构化包，智能体在推理时动态加载这些技能，已成为增强大语言模型（LLM）智能体的一种可靠机制。然而，推理时的技能增强存在根本性的局限：检索噪声会引入无关的指导，注入的技能内容会带来巨大的 Token 开销，且模型从未真正习得它仅仅是在遵循的知识。我们探究技能是否可以内化到模型参数中，从而在无需任何运行时技能检索的情况下实现零样本（zero-shot）自主行为。我们提出了 SKILL0，这是一个专为技能内化设计的上下文内强化学习框架。SKILL0 引入了一种训练时课程，从完整的技能上下文开始并逐步撤回。技能在离线状态下按类别分组，并与交互历史一起渲染成紧凑的视觉上下文，教导模型进行工具调用和多轮任务完成。随后，动态课程评估每个技能文件的同策略（on-policy）帮助程度，在策略线性衰减的预算内，仅保留当前策略仍能从中获益的技能，直到智能体在完全零样本的设置下运行。广泛的智能体实验表明，SKILL0 相比标准强化学习基线取得了显著提升（ALFWorld 提升 9.7%，Search-QA 提升 6.6%），同时保持了每步少于 0.5k Token 的高效上下文。我们的代码可在 https://github.com/ZJU-REAL/SkillZero 获取。

## Abstract
Agent skills, structured packages of procedural knowledge and executable resources that agents dynamically load at inference time, have become a reliable mechanism for augmenting LLM agents. Yet inference-time skill augmentation is fundamentally limited: retrieval noise introduces irrelevant guidance, injected skill content imposes substantial token overhead, and the model never truly acquires the knowledge it merely follows. We ask whether skills can instead be internalized into model parameters, enabling zero-shot autonomous behavior without any runtime skill retrieval. We introduce SKILL0, an in-context reinforcement learning framework designed for skill internalization. SKILL0 introduces a training-time curriculum that begins with full skill context and progressively withdraws it. Skills are grouped offline by category and rendered with interaction history into a compact visual context, teaching he model tool invocation and multi-turn task completion. A Dynamic Curriculum then evaluates each skill file's on-policy helpfulness, retaining only those from which the current policy still benefits within a linearly decaying budget, until the agent operates in a fully zero-shot setting. Extensive agentic experiments demonstrate that SKILL0 achieves substantial improvements over the standard RL baseline (+9.7\% for ALFWorld and +6.6\% for Search-QA), while maintaining a highly efficient context of fewer than 0.5k tokens per step. Our code is available at https://github.com/ZJU-REAL/SkillZero.

### 1. 核心问题与整体含义（研究动机和背景）
论文针对当前 LLM 智能体（Agent）主流的“推理时技能增强”范式提出了质疑。目前的智能体通常依赖从技能库中检索相关指令并注入 Prompt，这导致了三大问题：
*   **检索噪声：** 引入无关或误导性指导。
*   **Token 开销：** 技能描述增加了上下文长度，导致推理成本高昂且难以扩展。
*   **缺乏真实习得：** 模型只是在“遵循”指令而非“掌握”技能，能力停留在上下文层面而非参数层面。
**SKILL0 的核心目标**是实现“训练时有技能，推理时零技能”，通过强化学习将外部技能内化（Internalization）到模型参数中，使智能体具备零样本自主行为能力。

### 2. 方法论：核心思想与关键技术
SKILL0 提出了**上下文内强化学习（In-Context Reinforcement Learning, ICRL）**框架，其核心流程如下：

*   **视觉上下文渲染（Context Rendering）：** 为了降低 Token 压力，将文本交互历史和技能描述渲染成紧凑的视觉图像，并允许策略模型自适应生成压缩比 $c_t$。
*   **复合奖励函数：** 奖励函数不仅包含任务成功指标，还引入了对压缩效率的对数惩罚，鼓励模型在保持成功率的同时减少 Token 消耗。
*   **动态课程学习（Dynamic Curriculum）：** 这是实现内化的关键。
    1.  **相关性驱动分组：** 将技能按类别分组，并关联特定的验证子任务。
    2.  **帮助度评估（Helpfulness Evaluation）：** 定期评估当前策略在“有技能”和“无技能”情况下的性能差值 $\Delta_k$。
    3.  **线性预算衰减：** 设定一个随训练阶段线性减少的技能数量预算 $M(s)$。
    4.  **过滤与排序：** 仅保留 $\Delta_k > 0$ 且排序靠前的技能，直到预算归零，强制模型在没有外部辅助的情况下完成任务。

### 6. 主要结论与发现
*   **性能超越：** SKILL0 在零样本推理下，性能显著优于传统的 RL 基线（ALFWorld +9.7%，Search-QA +6.6%），甚至在不使用技能提示的情况下达到了与“技能增强型”方法（如 SkillRL）持平或更高的水平。
*   **极高效率：** 推理时每步 Token 消耗低于 0.5k，比 SkillRL 等方法降低了 5 倍以上。
*   **内化规律：** 技能的“帮助度”在训练中呈现“先升后降”的趋势——初期模型学习如何利用技能，后期随着能力内化，外部技能变得冗余。

### 7. 优点与亮点
*   **范式创新：** 首次将“技能内化”显式地定义为强化学习的训练目标，解决了智能体对 Prompt 的长期依赖问题。
*   **动态自适应：** 提出的动态课程不是死板的定时切换，而是基于模型当前能力的“帮助度”来决定技能的撤减，更加科学。
*   **多模态融合：** 利用视觉编码器处理长上下文，既保留了结构化信息又极大地压缩了 Token。

### 8. 不足与局限
*   **初始依赖：** SKILL0 的效果高度依赖于初始技能库（SkillBank）的质量，如果初始技能有误，可能会引导错误的内化。
*   **冷启动开销：** 离线技能分组和频繁的验证子任务评估（Helpfulness Evaluation）增加了训练前的准备工作和训练中的计算负担。
*   **领域迁移：** 当应用到全新领域时，需要重新定义技能类别和验证任务，自动化程度仍有提升空间。
