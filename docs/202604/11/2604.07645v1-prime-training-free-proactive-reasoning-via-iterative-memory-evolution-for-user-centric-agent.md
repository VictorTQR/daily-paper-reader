## 摘要
与人类用户协作处理复杂、长程任务的自主工具使用智能体的开发已成为智能体研究的前沿。在多轮人机交互过程中，用户需求的动态性和不确定性带来了重大挑战；智能体不仅必须调用工具，还必须通过有效的沟通迭代地完善对用户意图的理解。虽然强化学习的最新进展为构建更强大的工具使用智能体提供了路径，但现有方法需要昂贵的训练成本，并且在长程交互过程中难以处理轮次级的信用分配。为此，我们引入了 PRIME（通过迭代记忆演化实现的主动推理），这是一种无梯度学习框架，通过显式的经验积累而非昂贵的参数优化来实现智能体的持续演化。PRIME 将多轮交互轨迹提炼为结构化的、人类可读的经验，并将其组织在三个语义区域中：成功策略、失败模式和用户偏好。这些经验通过元级操作进行演化，并通过检索增强生成指导智能体未来的行为。我们在多个不同的以用户为中心的环境中进行的实验表明，PRIME 在提供成本效益和可解释性的同时，实现了与基于梯度的方法具有竞争力的性能。总之，PRIME 为构建主动、协作型智能体提供了一种实用的范式，使其能够从人机交互中学习，而无需承担基于梯度的训练所带来的计算负担。

## Abstract
The development of autonomous tool-use agents for complex, long-horizon tasks in collaboration with human users has become the frontier of agentic research. During multi-turn Human-AI interactions, the dynamic and uncertain nature of user demands poses a significant challenge; agents must not only invoke tools but also iteratively refine their understanding of user intent through effective communication. While recent advances in reinforcement learning offer a path to more capable tool-use agents, existing approaches require expensive training costs and struggle with turn-level credit assignment across extended interaction horizons. To this end, we introduce PRIME (Proactive Reasoning via Iterative Memory Evolution), a gradient-free learning framework that enables continuous agent evolvement through explicit experience accumulation rather than expensive parameter optimization. PRIME distills multi-turn interaction trajectories into structured, human-readable experiences organized across three semantic zones: successful strategies, failure patterns, and user preferences. These experiences evolve through meta-level operations and guide future agent behavior via retrieval-augmented generation. Our experiments across several diverse user-centric environments demonstrate that PRIME achieves competitive performance with gradient-based methods while offering cost-efficiency and interpretability. Together, PRIME presents a practical paradigm for building proactive, collaborative agents that learn from Human-AI interaction without the computational burden of gradient-based training.

*   **核心问题**：如何在不进行高昂梯度训练（如强化学习 RL）的情况下，提升 LLM 智能体在多轮人机交互中的“主动性”和“协作能力”？
*   **背景**：现有的 RL 方法（如 GRPO）虽然能提升智能体性能，但面临三大痛点：1) 算力成本极高；2) 学习到的权重具有模型依赖性，无法跨架构迁移；3) 决策过程黑盒化，缺乏可解释性。
*   **研究意义**：在以用户为中心的任务中，用户意图往往是隐性的，智能体需要通过提问、澄清和总结来逐步挖掘意图。PRIME 旨在通过“经验积累”而非“参数优化”来解决这一挑战。

### 2. 方法论：PRIME 框架
PRIME 的核心思想是将智能体的提升视为一个**知识演化问题**。其流程包含四个关键阶段：
*   **探索（Exploration）**：智能体在环境中进行多轮交互，收集原始轨迹（包含状态、动作、奖励和观察）。
*   **提炼（Distillation）**：
    *   **信用分配**：利用 Reward-to-Go (R2G) 或均等分配法识别轨迹中的关键轮次（Key Turns）。
    *   **结构化存储**：将轨迹转化为包含“情境、动作、结果、教训”的结构化记忆。
    *   **语义分区**：根据奖励将记忆划分为**黄金区**（成功策略）、**警告区**（失败模式）和**偏好区**（用户行为模式）。
*   **演化（Evolution）**：通过元级操作（Meta-level operations）优化记忆库，包括：
    *   **变异**：根据反馈细化经验描述。
    *   **泛化**：去除特定环境细节，提取通用知识。
    *   **交叉**：结合多个经验的见解。
    *   **剪枝**：移除低频或表现不佳的记忆。
*   **推理（Inference）**：采用**阶段感知检索**。根据当前交互阶段（探索、验证、完成）检索最相关的记忆，通过 RAG（检索增强生成）引导智能体生成响应。

### 6. 主要结论与发现
*   **性能卓越**：PRIME 在所有 8 个环境中均显著提升了开源基座模型的表现，在某些任务（如 FunctionGym）中提升幅度高达 193%。
*   **逼近 RL 效果**：虽然在绝对性能上略逊于经过深度 RL 训练的模型，但 PRIME 以极低的成本达到了极具竞争力的水平。

*   **阶段感知的重要性**：在交互的不同阶段（如初期探索 vs 后期总结）提供针对性的记忆指导是提升效率的关键。

### 7. 优点与亮点
*   **免训练（Training-Free）**：无需调整模型参数，适合无法获取权重或算力受限的场景。
*   **高可解释性**：记忆库以人类可读的自然语言存储，用户可以清晰看到智能体“学到了什么”。
*   **即插即用**：经验库可以像插件一样在不同模型间流转。
*   **负面反馈利用**：通过“警告区”显式告知智能体避开过去的错误，这比单纯的正向引导更有效。

### 8. 不足与局限
*   **依赖高级模型进行提炼**：PRIME 的效果高度依赖于负责提炼和演化的“元智能体”（如 GPT-4o）的质量。
*   **检索开销**：每一轮交互都需要进行检索和 Prompt 扩充，可能会增加推理延迟和 Token 消耗。
*   **上限瓶颈**：由于不改变模型权重，其性能上限受限于基座模型理解和执行复杂指令的基础能力。
*   **长程记忆管理**：随着交互增加，记忆库可能变得臃肿，虽然有剪枝机制，但在极大规模任务下的维护成本尚待观察。
