## 摘要
深度研究智能体 (DRA) 将大语言模型 (LLM) 的推理能力与外部工具相结合。记忆系统使 DRA 能够利用历史经验，这对于高效推理和自主演化至关重要。现有方法依赖于从记忆中检索相似轨迹以辅助推理，但面临记忆演化效率低下以及存储和检索成本不断增加的关键局限。为解决这些问题，我们提出了一种新型的记忆智能体 (MIA) 框架，该框架由“管理者-规划者-执行者” (Manager-Planner-Executor) 架构组成。记忆管理者是一个非参数化记忆系统，可以存储压缩后的历史搜索轨迹。规划者是一个参数化记忆智能体，能够为问题生成搜索计划。执行者是另一个智能体，可以在搜索计划的指导下搜索并分析信息。为了构建 MIA 框架，我们首先采用交替强化学习范式来增强规划者与执行者之间的协作。此外，我们使规划者能够在测试时学习 (test-time learning) 过程中持续演化，在不中断推理过程的情况下，随推理同步进行即时更新。此外，我们在参数化记忆与非参数化记忆之间建立了双向转换循环，以实现高效的记忆演化。最后，我们引入了反思和无监督判断机制，以提升在开放世界中的推理和自我演化能力。在 11 个基准测试上的广泛实验证明了 MIA 的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

### 1. 核心问题与研究动机
*   **研究背景**：深度研究智能体（Deep Research Agents, DRAs）通过整合 LLM 推理与外部工具（如搜索引擎）来处理复杂任务。记忆系统是其积累经验、优化策略的关键。
*   **核心问题**：现有的智能体记忆系统存在四大局限：
    1.  **注意力稀释**：长上下文记忆会导致模型注意力分散。
    2.  **噪声干扰**：无关或弱相关的历史记录会降低推理质量。
    3.  **存储瓶颈**：持续增长的上下文历史带来巨大的存储压力。
    4.  **检索低效**：在大规模记忆库中检索会导致计算成本和延迟增加。
*   **研究目标**：开发一种能够高效存储、检索并实现“自我演化”的记忆框架，将历史经验转化为指导未来规划的知识，而非简单的文本堆砌。

### 2. 核心方法论
MIA 提出了 **Manager-Planner-Executor（管理者-规划者-执行者）** 架构，核心思想是解耦历史记忆、参数化规划和动态执行。

*   **Manager（记忆管理者）**：非参数化存储系统。它将冗长的搜索轨迹压缩为结构化的“工作流摘要”（Workflow Summaries），并根据语义相似性、价值奖励（成功率）和频率奖励进行混合检索。
*   **Planner（规划者）**：参数化记忆智能体。负责分析检索到的案例，生成分步搜索计划，并具备“反思-重规划”机制。
*   **Executor（执行者）**：操作终端。负责解析规划者的指令，通过 ReAct 循环调用工具并反馈执行状态。
*   **关键技术细节**：
    *   **两阶段交替强化学习（RL）**：基于 GRPO 算法。第一阶段训练执行者遵循计划；第二阶段在冻结执行者的基础上，训练规划者吸收记忆并优化规划。
    *   **测试时学习（Test-Time Learning, TTL）**：在推理过程中同步更新规划者的参数（参数化记忆）并提取新工作流存入管理者（非参数化记忆），实现即时演化。
    *   **无监督自我演化**：引入类似学术评审的“审稿人-区域主席”机制，通过逻辑、事实、格式三个维度的 LLM 评审替代 Ground-Truth 标签，支持在无标注环境下的持续进化。

### 3. 实验设计
*   **数据集/场景**：涵盖了 11 个基准测试。

    *   **纯文本任务**：HotpotQA, 2WikiMultiHopQA, SimpleQA, GAIA。
*   **对比方法**：
    *   **前沿模型**：GPT-4o, GPT-5.4, Gemini-3-Flash, Claude-Sonnet-4.6。
    *   **搜索智能体**：MMSearch-R1, DeepMMSearch-R1, WebWatcher, Deepeyes2。
    *   **记忆框架**：RAG, Mem0, A-Mem, ReasoningBank, ExpeL, Memento。

### 4. 资源与算力
*   **硬件配置**：使用了 **8 张 GPU**（未指明具体型号，但根据 veRL 框架和模型规模推测为 A100 或 H100 级别）。
*   **模型初始化**：执行者基于 Qwen2.5-VL-7B，规划者基于 Qwen3-8B，记忆管理者/评判者使用 Qwen3-32B。
*   **训练参数**：执行者训练 8 个 epoch，规划者训练 4 个 epoch；TTL 阶段学习率为 $1 \times 10^{-6}$，每个样本进行 4 次 rollout。

### 5. 实验数量与充分性
*   **实验规模**：论文进行了大规模的跨领域测试，涵盖了从基础事实问答到复杂多跳推理的多种场景。
*   **消融实验**：非常充分。分别验证了“仅记忆”、“仅规划”、“反思机制”、“强化学习训练”以及“TTL”对最终性能的贡献。
*   **客观性**：通过对比不同规模的开源模型和闭源 SOTA 模型，证明了 MIA 框架的普适性。特别是展示了 7B 规模的 MIA 执行者在多项指标上超越了 32B 甚至更大的闭源模型，体现了架构的优越性而非单纯堆砌参数。

### 6. 主要结论与发现
*   **性能飞跃**：MIA 显著提升了现有 SOTA 模型的表现。例如，将 GPT-5.4 在 LiveVQA 上的表现提升了 9%。
*   **小模型逆袭**：基于 Qwen2.5-VL-7B 的 MIA 在平均性能上超过了 Qwen2.5-VL-32B 约 18%。
*   **记忆有效性**：传统的长上下文记忆（如 RAG, Mem0）在复杂工具交互中往往表现不如“无记忆”基准，而 MIA 的结构化规划记忆成功克服了这一噪声问题。
*   **持续进化**：在无监督设置下，MIA 通过多次迭代（Epoch 1-3）表现出稳定的性能增长，证明了其在开放世界中自我学习的潜力。

### 7. 优点与亮点
*   **架构创新**：Manager-Planner-Executor 的三层架构有效解决了记忆冗余与推理效率的矛盾。
*   **双向记忆转换**：将非参数化的经验实时转化为参数化的能力（通过 TTL），这种“内化”机制非常符合生物脑的学习逻辑。
*   **无监督机制**：提出的“类同行评审”评估框架为智能体在缺乏人类反馈的环境下演化提供了新思路。
*   **泛化性强**：实验证明该框架不仅适用于开源模型，也能作为“插件”提升闭源商业模型的性能。

### 8. 不足与局限
*   **计算开销**：虽然 TTL 提升了性能，但在推理时进行参数更新和多次 rollout 会显著增加推理延迟和算力消耗。
*   **工具集相对简单**：实验中主要使用了基础的文本和图像搜索工具，对于更复杂的 API 调用或多工具协同的覆盖尚待加强。
*   **模型依赖**：框架的效果高度依赖于作为“评判者”和“管理者”的高阶模型（如 Qwen3-32B）的质量，若核心模型存在偏见，可能会影响演化方向。
