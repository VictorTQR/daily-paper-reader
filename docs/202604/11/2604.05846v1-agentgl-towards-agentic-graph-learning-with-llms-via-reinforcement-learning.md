## 摘要
大语言模型（LLMs）日益依赖智能体能力（如迭代检索、工具使用和决策制定）来克服静态参数化知识的局限性。然而，现有的智能体框架将外部信息视为非结构化文本，未能利用现实世界数据中固有的拓扑依赖关系。为了填补这一空白，我们引入了智能体图学习（Agentic Graph Learning, AGL），这一范式将图学习重新定义为拓扑感知导航与基于 LLM 推理的交替过程。具体而言，我们提出了 AgentGL，这是首个针对 AGL 的强化学习（RL）驱动框架。AgentGL 为 LLM 智能体配备了用于多尺度探索的图原生工具，通过搜索约束思维来规范工具使用以平衡准确性与效率，并采用图条件课程强化学习策略，在无需逐步监督的情况下稳定长程策略学习。在多种文本属性图（TAG）基准测试和多个 LLM 主干模型上，AgentGL 的表现显著优于强力的 GraphLLM 和 GraphRAG 基准模型，在节点分类和链路预测中分别实现了高达 17.5% 和 28.4% 的绝对提升。这些结果表明，AGL 是使 LLM 能够在复杂关系环境中自主导航和推理的一个极具前景的前沿方向。代码已在 https://github.com/sunyuanfu/AgentGL 公开。

## Abstract
Large Language Models (LLMs) increasingly rely on agentic capabilities-iterative retrieval, tool use, and decision-making-to overcome the limits of static, parametric knowledge. Yet existing agentic frameworks treat external information as unstructured text and fail to leverage the topological dependencies inherent in real-world data. To bridge this gap, we introduce Agentic Graph Learning (AGL), a paradigm that reframes graph learning as an interleaved process of topology-aware navigation and LLM-based inference. Specifically, we propose AgentGL, the first reinforcement learning (RL)-driven framework for AGL. AgentGL equips an LLM agent with graph-native tools for multi-scale exploration, regulates tool usage via search-constrained thinking to balance accuracy and efficiency, and employs a graph-conditioned curriculum RL strategy to stabilize long-horizon policy learning without step-wise supervision. Across diverse Text-Attributed Graph (TAG) benchmarks and multiple LLM backbones, AgentGL substantially outperforms strong GraphLLMs and GraphRAG baselines, achieving absolute improvements of up to 17.5% in node classification and 28.4% in link prediction. These results demonstrate that AGL is a promising frontier for enabling LLMs to autonomously navigate and reason over complex relational environments. The code is publicly available at https://github.com/sunyuanfu/AgentGL.

### 2. 方法论：核心思想与技术细节
AgentGL 是首个由强化学习（RL）驱动的 AGL 框架，其核心由三个关键部分组成：

*   **图原生搜索工具（GNS Tools）**：
    *   **多尺度探索**：提供 1-hop/2-hop 邻域搜索（局部）、结构显著性搜索（基于 PageRank 的全局拓扑先验）和图稠密搜索（基于语义相似度的全局搜索）。
    *   **加权融合排名**：利用查询相关性和目标实例特征的融合嵌入对搜索结果进行排名。
*   **两阶段强化学习策略**：
    *   **阶段 1：策略引导（Bootstrapping）**：通过组合奖励（格式奖励 + 准确性奖励 + 工具覆盖奖励）训练 LLM 掌握工具使用规范，防止模型陷入不搜索或只用单一工具的局部最优。
    *   **阶段 2：减少过度搜索（MSO）**：引入“搜索约束思维”，通过回顾性终止触发器（强制模型在搜索后评估证据充分性）和认知密度正则化（惩罚肤浅推理），引导模型以最少的搜索步骤达成准确预测。
*   **图条件课程学习（GCCL）**：
    *   **难度量化**：利用图的拓扑属性（如节点度、局部同质性）和语义先验（语义与标签的一致性）自动计算样本难度。
    *   **由易到难训练**：稳定长程策略的训练过程，加速收敛并提高泛化能力。

*   **实验规模**：实验非常充分。不仅在 7 个数据集上进行了双任务测试，还跨越了 3B 和 7B 两种模型参数规模。

    *   验证了奖励函数中各组件（覆盖奖励、认知密度、回顾触发器）的作用。

### 6. 主要结论与发现
*   **性能飞跃**：AgentGL 在节点分类上最高提升 **17.5%**，在链路预测上最高提升 **28.4%**。
*   **自适应性**：相比静态上下文填充（GraphLLM）或简单的检索增强（GraphRAG），AgentGL 的交替推理循环能更有效地抑制噪声并获取关键证据。
*   **效率平衡**：通过第二阶段训练，模型在保持高精度的同时，减少了约 **17.5%** 的冗余工具调用。
*   **规模效应**：随着主干模型从 3B 扩展到 7B，AgentGL 的智能体导航能力和零样本泛化能力显著增强。

### 7. 优点（亮点）
*   **范式创新**：首次将图学习定义为智能体决策过程，打破了图数据处理的静态限制。
*   **训练稳定性**：通过课程学习和两阶段奖励设计，解决了 LLM 在缺乏步骤级监督时难以学习长程图搜索策略的难题。
*   **工具完备性**：设计的工具箱兼顾了局部拓扑与全局语义，覆盖了图数据的多尺度特征。

### 8. 不足与局限
*   **模态限制**：目前仅支持文本属性图（TAG），尚未扩展到包含图像或视频等多模态属性的图数据。
*   **数据分配敏感性**：MSO 阶段的稳定性依赖于两个训练阶段之间数据分配的精确平衡。
*   **图密度挑战**：在极稠密的图中，搜索工具可能会返回过多噪声，其扩展性仍有待进一步探索。
*   **推理成本**：尽管优化了搜索次数，但相比单次推理的 GraphLLM，多轮交互的智能体模式仍存在更高的推理延迟。
