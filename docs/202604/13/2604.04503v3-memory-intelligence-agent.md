## 摘要
深度研究智能体 (DRA) 将大语言模型 (LLM) 的推理能力与外部工具相结合。记忆系统使 DRA 能够利用历史经验，这对于高效推理和自主演化至关重要。现有方法依赖于从记忆中检索相似轨迹来辅助推理，但面临记忆演化效率低下以及存储和检索成本不断增加的关键局限。为解决这些问题，我们提出了一种新型的记忆智能体 (MIA) 框架，该框架由“管理者-规划者-执行者” (Manager-Planner-Executor) 架构组成。记忆管理者是一个非参数化记忆系统，可以存储压缩后的历史搜索轨迹。规划者是一个参数化记忆智能体，能够为问题生成搜索计划。执行者是另一个智能体，可以在搜索计划的指导下搜索并分析信息。为了构建 MIA 框架，我们首先采用交替强化学习范式来增强规划者与执行者之间的协作。此外，我们使规划者能够在测试时学习过程中持续演化，在不中断推理过程的情况下，随推理同步进行实时更新。此外，我们在参数化记忆与非参数化记忆之间建立了双向转换循环，以实现高效的记忆演化。最后，我们引入了反思和无监督判断机制，以提升在开放世界中的推理和自我演化能力。在 11 个基准测试上的广泛实验证明了 MIA 的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

#### 1. 论文的核心问题与整体含义
*   **研究动机**：深度研究智能体（Deep Research Agents, DRAs）通过集成大语言模型（LLM）推理与外部工具来解决复杂任务。然而，现有的智能体记忆系统存在四大痛点：
    1.  **注意力稀释**：长上下文记忆会分散模型注意力。
    2.  **噪声干扰**：无关或弱相关的历史信息会降低推理质量。
    3.  **存储瓶颈**：随着交互增加，存储成本呈指数级增长。
    4.  **知识类型局限**：现有记忆多关注“事实性知识”（是什么），而忽略了“过程性知识”（怎么做，如搜索策略、失败教训）。
*   **核心目标**：提出一种能够高效存储、检索并实现自主演化的记忆框架，使智能体能从历史经验中提取策略，而非仅仅堆砌文本。

#### 2. 论文提出的方法论
MIA 采用了**管理者-规划者-执行者（Manager-Planner-Executor）**的三层解耦架构：
*   **Memory Manager（管理者）**：非参数化记忆系统。它将冗长的搜索轨迹压缩为结构化的“工作流摘要”（Workflow Summaries），并基于语义相似度、价值奖励（成功率）和频率奖励进行混合检索。
*   **Planner（规划者）**：参数化记忆智能体。负责分析检索到的记忆，生成分步搜索计划。它支持**测试时学习（Test-Time Learning, TTL）**，即在推理过程中通过在线强化学习实时更新参数。
*   **Executor（执行者）**：操作终端。负责解析计划并利用 ReAct 范式调用外部工具（如搜索引擎、图像搜索）。
*   **关键技术细节**：
    *   **交替强化学习（Alternating RL）**：基于 GRPO 算法，第一阶段训练执行者遵循计划，第二阶段训练规划者吸收记忆并进行反思。
    *   **双向转换循环**：将成功的轨迹存入非参数记忆（Manager），同时通过 TTL 将其内化为规划者的参数（Planner）。
    *   **无监督判断机制**：在缺乏 Ground-truth 的开放世界中，模拟学术评审流程（逻辑、事实、有效性三个评审员 + 领域主席裁决），实现自我演化。

#### 6. 论文的主要结论与发现

*   **小胜大**：搭载 MIA 框架的 **Qwen2.5-VL-7B** 执行者，平均性能超过了参数量大得多的 **Qwen2.5-VL-32B** 约 18%。

#### 7. 优点
*   **架构创新**：将记忆管理、高层规划与底层执行解耦，有效解决了长上下文带来的噪声问题。
*   **在线进化**：TTL 机制允许智能体在推理时“边做边学”，无需重新进行大规模离线训练。
*   **多模态融合**：不仅支持文本搜索，还深度集成了图像到图像的搜索与理解，适应深度研究需求。
*   **无监督能力**：提出的“评审员-主席”机制为缺乏标注数据的智能体进化提供了新思路。

#### 8. 不足与局限
*   **计算开销**：TTL 机制在推理时需要进行多次采样（Rollouts）和参数更新，这会显著增加推理延迟和算力成本。
*   **依赖 Judger 质量**：无监督演化和 RL 训练高度依赖于一个更强大的“裁判模型”（如文中使用的 Qwen3-32B），若裁判模型存在偏见或幻觉，会引导智能体走向错误方向。
*   **工具集复杂度**：虽然在搜索任务上表现优异，但对于更复杂的工业级工具（如代码执行、数据库操作）的适配性尚待验证。
