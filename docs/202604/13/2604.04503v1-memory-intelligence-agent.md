## 摘要
深度研究智能体（DRAs）将大语言模型（LLM）推理与外部工具相结合。记忆系统使 DRAs 能够利用历史经验，这对于高效推理和自主演化至关重要。现有方法依赖于从记忆中检索相似轨迹来辅助推理，但面临记忆演化效率低下以及存储和检索成本不断增加的关键局限。为了解决这些问题，我们提出了一种新型的记忆智能体（MIA）框架，该框架由“管理者-规划者-执行者”架构组成。记忆管理者是一个非参数化记忆系统，可以存储压缩后的历史搜索轨迹。规划者是一个参数化记忆智能体，能够为问题生成搜索计划。执行者是另一个智能体，可以在搜索计划的指导下搜索和分析信息。为了构建 MIA 框架，我们首先采用交替强化学习范式来增强规划者与执行者之间的协作。此外，我们使规划者能够在测试时学习过程中持续演化，在不中断推理过程的情况下随推理同步进行实时更新。此外，我们在参数化记忆和非参数化记忆之间建立了双向转换循环，以实现高效的记忆演化。最后，我们引入了反思和无监督判断机制，以提升在开放世界中的推理和自我演化能力。在 11 个基准测试上的广泛实验证明了 MIA 的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

深度研究智能体（Deep Research Agents, DRAs）通过结合 LLM 推理与外部工具（如搜索引擎）来解决复杂任务。然而，现有的智能体记忆系统存在以下痛点：
*   **长上下文瓶颈**：直接存储原始搜索轨迹会导致上下文过长，稀释模型注意力，引入噪声，并增加存储与检索的计算成本。
*   **记忆演化效率低**：现有方法多为静态检索，缺乏在推理过程中实时自我更新和进化的能力。
*   **规划与执行脱节**：缺乏针对特定任务的协同训练，导致规划者（Planner）生成的计划不切实际，或执行者（Executor）无法准确遵循指令。
*   **无监督环境下的局限**：在缺乏真实标签（Ground Truth）的开放世界中，智能体难以进行自我评估和持续进化。

### 2. 方法论：核心思想与关键技术
MIA 提出了一个**管理者-规划者-执行者（Manager-Planner-Executor）**架构，核心思想是将记忆解耦为“非参数化”和“参数化”两部分：
*   **架构组成**：
    *   **记忆管理者 (Manager)**：非参数化系统，负责将冗长的搜索轨迹压缩为结构化的“工作流摘要”，并基于语义相似度、价值奖励和频率奖励进行高效检索。
    *   **规划者 (Planner)**：参数化记忆智能体，负责分析历史经验并生成分步搜索计划。
    *   **执行者 (Executor)**：负责具体工具调用（ReAct 模式）并反馈执行状态。
*   **关键技术细节**：
    *   **交替强化学习 (Alternating RL)**：基于 GRPO 算法，第一阶段训练执行者遵循计划，第二阶段训练规划者吸收记忆并进行反思（Reflect-Replan）。
    *   **测试时学习 (Test-Time Learning, TTL)**：在推理过程中同步更新规划者的参数（参数化记忆）并存入新工作流（非参数化记忆），实现“边做边学”。
    *   **无监督评审机制**：模拟学术论文评审，通过三个专业评审（逻辑、事实、结果有效性）和一个领域主席（AC）进行元决策，在无标签情况下提供伪监督信号。

### 6. 主要结论与发现

*   **以小博大**：搭载 MIA 的 **Qwen2.5-VL-7B** 平均性能提升 31%，甚至超过了未搭载该框架的 32B 模型（领先 18%）。

### 7. 优点与亮点
*   **双重记忆循环**：巧妙地结合了非参数化记忆（显式案例）和参数化记忆（隐式模型权重更新），解决了存储成本与推理效率的矛盾。
*   **测试时在线更新**：无需中断推理即可进行参数微调，这在实时智能体应用中具有极高的实用价值。
*   **无监督评审框架**：引入“评审员-主席”机制，为解决 LLM 幻觉和无标签环境下的自我评估提供了新思路。

### 8. 不足与局限
*   **计算开销**：虽然减少了上下文长度，但测试时学习（TTL）涉及在线梯度更新和多次 Rollout，会显著增加单次推理的时间延迟和算力消耗。
*   **工具集相对简单**：实验中主要使用了基础的文本和图像搜索工具，对于更复杂的 API 调用或动态环境交互的泛化能力仍需进一步验证。
*   **模型依赖**：框架的效果高度依赖于作为“管理者”和“评审员”的高阶模型（如 Qwen3-32B）的判断准确性。
