## 摘要
深度研究智能体 (DRA) 将大语言模型 (LLM) 推理与外部工具相结合。记忆系统使 DRA 能够利用历史经验，这对于高效推理和自主演化至关重要。现有方法依赖于从记忆中检索相似轨迹来辅助推理，但面临记忆演化效率低下以及存储和检索成本增加的关键局限。为了解决这些问题，我们提出了一种新型的记忆智能体 (MIA) 框架，该框架由“管理者-规划者-执行者” (Manager-Planner-Executor) 架构组成。记忆管理者是一个非参数化记忆系统，可以存储压缩后的历史搜索轨迹。规划者是一个参数化记忆智能体，能够针对问题生成搜索计划。执行者是另一个智能体，可以在搜索计划的指导下搜索并分析信息。为了构建 MIA 框架，我们首先采用交替强化学习范式来增强规划者与执行者之间的协作。此外，我们使规划者能够在测试时学习过程中持续演化，在不中断推理过程的情况下，随推理同步进行实时更新。此外，我们在参数化记忆与非参数化记忆之间建立了双向转换循环，以实现高效的记忆演化。最后，我们引入了反思和无监督判断机制，以提升在开放世界中的推理和自我演化能力。在 11 个基准测试上的广泛实验证明了 MIA 的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

*   **研究动机**：深度研究智能体（Deep Research Agents, DRAs）在处理复杂任务时，依赖记忆系统来积累经验。然而，现有的记忆机制存在三大痛点：
    1.  **效率低下**：长上下文记忆会稀释模型注意力，引入无关噪声。
    2.  **成本高昂**：随着任务增加，存储和检索大规模原始轨迹的计算成本剧增。
    3.  **演化困难**：现有方法多为静态检索，缺乏在推理过程中实时自我优化和从失败中学习的能力。
*   **核心目标**：提出一种名为 **MIA** 的框架，通过解耦记忆管理、规划与执行，实现高效的记忆演化和长程推理。

### 2. 方法论：核心思想与关键技术
MIA 采用了**管理者-规划者-执行者（Manager-Planner-Executor）**的脑启发式架构：
*   **核心架构**：
    *   **Memory Manager（管理者）**：非参数化记忆系统。负责将冗长的搜索轨迹压缩为结构化的“工作流（Workflow）”摘要，存储成功（正向）和失败（负向）的案例。
    *   **Planner（规划者）**：参数化记忆智能体（基于 Qwen3-8B）。负责分析历史案例并为当前问题制定分步搜索计划。
    *   **Executor（执行者）**：操作终端（基于 Qwen2.5-VL-7B）。负责根据计划调用工具（如搜索引擎、图像搜索）并进行推理。
*   **关键技术细节**：
    *   **交替强化学习（Alternating RL）**：使用 **GRPO（群组相对策略优化）** 算法。第一阶段训练执行者遵循计划；第二阶段在冻结执行者的基础上，训练规划者吸收记忆并进行反思。
    *   **测试时学习（Test-Time Learning, TTL）**：在推理阶段，规划者通过在线梯度更新实时调整参数，实现“边推理边进化”，无需中断工作流。
    *   **双向记忆转换**：建立非参数化（外部存储）与参数化（模型权重）记忆的循环，将经验内化。
    *   **无监督评审机制（Reviewer-AC）**：针对无标注环境，设计了模拟学术评审的机制，由三个专门的评审智能体（逻辑、事实、有效性）和一个领域主席（AC）共同判断推理质量，驱动自主演化。

*   **训练细节**：执行者在 FVQA-train 上训练 8 个 epoch；规划者在混合数据集上训练 4 个 epoch。推理时使用 vLLM 引擎。

### 5. 实验数量与充分性
*   **实验规模**：进行了跨越 11 个数据集的大规模测试，涵盖了从 7B 到 32B 不同参数规模的模型。
*   **消融实验**：非常充分。分别验证了“仅记忆”、“仅规划”、“反思机制”、“强化学习训练”以及“TTL”对最终性能的贡献。
*   **客观性**：通过引入第三方 LLM（Qwen3-32B）作为公正评判者（LLM Judger），并对比了闭源模型在相同工具环境下的表现，确保了对比的公平性。

### 6. 主要结论与发现
*   **性能飞跃**：MIA 显著提升了现有最强模型的上限。例如，将 GPT-5.4 在 LiveVQA 上的表现提升了 **9%**。
*   **以小博大**：搭载 MIA 的 **7B 执行者** 平均性能提升 31%，甚至超越了未搭载该框架的 **32B 模型**（领先 18%）。
*   **自主演化有效**：在无监督设置下，MIA 通过自我评审机制实现了与有监督设置相近的性能，且随着迭代次数增加，准确率稳步上升（如 59.6% → 63.2%）。
*   **工具调用优化**：分析显示 MIA 能更有效地利用工具，避免了无记忆模型常见的重复搜索或过早放弃的问题。

### 7. 优点与亮点
*   **架构创新**：解耦了规划与执行，通过参数化更新解决了长上下文记忆的噪声问题。
*   **实时进化**：TTL 机制允许智能体在测试阶段根据环境反馈即时优化，这在动态研究任务中极具价值。
*   **无监督能力**：提出的 Reviewer-AC 评审框架为智能体在缺乏人类反馈的开放世界中自我进化提供了可行路径。

### 8. 不足与局限
*   **计算开销**：虽然 TTL 提升了性能，但在推理过程中进行参数更新会增加额外的计算负担和延迟。
*   **环境依赖**：目前的实验主要集中在搜索和问答类任务，对于需要复杂物理交互或极长程规划的动态环境（如机器人控制）尚需进一步验证。
*   **模型依赖**：框架的效果仍受限于基础模型（如 Qwen 系列）的底层能力，对于极小规模模型的带动作用可能有限。
