## 摘要
大语言模型（LLMs）日益依赖智能体能力（如迭代检索、工具使用和决策制定）来克服静态参数化知识的局限性。然而，现有的智能体框架将外部信息视为非结构化文本，未能利用现实世界数据中固有的拓扑依赖关系。为了填补这一空白，我们引入了智能体图学习（Agentic Graph Learning, AGL），这一范式将图学习重新定义为拓扑感知导航与基于 LLM 推理的交替过程。具体而言，我们提出了 AgentGL，这是首个针对 AGL 的强化学习（RL）驱动框架。AgentGL 为 LLM 智能体配备了用于多尺度探索的图原生工具，通过搜索约束思维来规范工具使用以平衡准确性与效率，并采用图条件课程强化学习策略，在无需逐步监督的情况下稳定长程策略学习。在多种文本属性图（TAG）基准测试和多个 LLM 主干模型上，AgentGL 的表现显著优于强力的 GraphLLM 和 GraphRAG 基准模型，在节点分类和链路预测任务中分别实现了高达 17.5% 和 28.4% 的绝对提升。这些结果表明，AGL 是使 LLM 能够在复杂关系环境中自主导航和推理的一个极具前景的前沿方向。代码已在 https://github.com/sunyuanfu/AgentGL 公开。

## Abstract
Large Language Models (LLMs) increasingly rely on agentic capabilities-iterative retrieval, tool use, and decision-making-to overcome the limits of static, parametric knowledge. Yet existing agentic frameworks treat external information as unstructured text and fail to leverage the topological dependencies inherent in real-world data. To bridge this gap, we introduce Agentic Graph Learning (AGL), a paradigm that reframes graph learning as an interleaved process of topology-aware navigation and LLM-based inference. Specifically, we propose AgentGL, the first reinforcement learning (RL)-driven framework for AGL. AgentGL equips an LLM agent with graph-native tools for multi-scale exploration, regulates tool usage via search-constrained thinking to balance accuracy and efficiency, and employs a graph-conditioned curriculum RL strategy to stabilize long-horizon policy learning without step-wise supervision. Across diverse Text-Attributed Graph (TAG) benchmarks and multiple LLM backbones, AgentGL substantially outperforms strong GraphLLMs and GraphRAG baselines, achieving absolute improvements of up to 17.5% in node classification and 28.4% in link prediction. These results demonstrate that AGL is a promising frontier for enabling LLMs to autonomously navigate and reason over complex relational environments. The code is publicly available at https://github.com/sunyuanfu/AgentGL.

*   **核心问题**：如何构建一个能够自主、动态地在图结构中导航、积累结构证据并迭代优化搜索轨迹的智能体系统？

### 2. 方法论：AgentGL 框架
论文提出了 **AgentGL**，这是首个由强化学习（RL）驱动的智能体图学习（AGL）框架。

*   **核心思想**：将图学习建模为一个“推理-行动-观察”的循环过程，通过 RL 优化智能体的搜索策略。
*   **关键技术细节**：
    1.  **图原生搜索工具（GNS Tools）**：
        *   **1-hop/2-hop 搜索**：基于拓扑邻域和语义相关性融合评分进行局部探索。
        *   **结构显著性搜索（SS）**：利用 PageRank 等全局指标识别关键节点。
        *   **图密集搜索（DENSE）**：利用向量相似度跨越拓扑断裂寻找语义相关节点。
    2.  **两阶段强化学习策略**：
        *   **阶段 1：策略引导（Bootstrapping）**：通过奖励塑造（格式奖励、准确性奖励、工具覆盖奖励）教会模型如何使用图工具。
        *   **阶段 2：缓解搜索过度（MSO）**：引入“搜索约束思维”，通过回顾性终止触发和认知密度正则化，惩罚冗余搜索，鼓励深度推理。
    3.  **图条件课程学习（GCCL）**：
        *   根据图的拓扑属性（如节点度、同质性）和语义先验自动评估样本难度，实现从易到难的平滑训练。

### 3. 实验设计

*   **实验规模**：实验设计非常详尽。除了在 7 个数据集上进行主实验外，还进行了跨任务（NC/LP）、跨模型规模（3B/7B）、跨 RL 算法（GRPO/REINFORCE++）的对比。

### 6. 主要结论与发现
*   **性能卓越**：AgentGL 在节点分类上最高提升 17.5%，在链路预测上最高提升 28.4%。
*   **动态优于静态**：交替的“搜索-推理”循环显著优于静态的上下文注入（GraphLLM）或简单的检索增强（GraphRAG）。
*   **效率与精度平衡**：通过第二阶段的 MSO 训练，模型在减少约 17.5% 工具调用的同时，准确率反而有所提升，证明了“少即是多”的搜索原则。
*   **规模效应**：随着主干模型从 3B 扩展到 7B，智能体的图导航能力和零样本迁移能力显著增强。

### 7. 优点（亮点）
*   **范式创新**：首次系统性地提出了“智能体图学习（AGL）”概念，并给出了完整的 RL 解决方案。
*   **工具设计完备**：GNS 工具包兼顾了局部与全局、拓扑与语义，覆盖了图探索的各个维度。
*   **训练稳定性**：引入图条件课程学习（GCCL），利用图的固有属性（如度数、同质性）量化难度，有效解决了长程策略学习难以收敛的问题。

### 8. 不足与局限
*   **模态限制**：目前仅支持文本属性图，尚未扩展到包含图像、视频等多模态属性的图数据。
*   **训练成本**：虽然推理时效率提升，但 RL 训练过程（尤其是多阶段训练和大量采样）对算力要求较高。
*   **图密度敏感性**：在极稠密图上的表现尚未充分讨论，可能会面临搜索空间爆炸或噪声过多的挑战。
*   **数据分配依赖**：两阶段训练之间的数据分配比例对最终性能有较大影响，需要精细调优。
